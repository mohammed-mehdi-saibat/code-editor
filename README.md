# Web Code Editor

A lightweight, browser-based code editor built with **HTML**, **CSS**, and **JavaScript**. This project allows users to write, edit, and preview HTML, CSS, and JavaScript code in real-time.

## Features

- **Real-time Preview**: Instantly see the results of your HTML, CSS, and JavaScript code.
- **Separate Panels**: Work with HTML, CSS, and JavaScript in dedicated sections.
- **Clean Interface**: Simple and easy-to-use layout for coding efficiently.
- **No Setup Required**: Just open in a browser and start coding.
